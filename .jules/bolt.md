## 2026-02-19 - State Updates in Composition | Learning: Updating state directly within the composition body (e.g., inside a Composable function instead of LaunchedEffect) can cause unpredictable behavior and unnecessary recompositions. | Action: Always use LaunchedEffect or SideEffect for state updates triggered by composition.
